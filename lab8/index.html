Основні техніки оптимізації зображень

Вибір правильного формату

WebP для фотографій, SVG для векторної графіки, PNG для прозорості.

Адаптивні зображення

Використання srcset, sizes та picture для оптимізації під різні пристрої.

Ліниве завантаження

Зображення завантажуються тільки коли вони наближаються до видимої області.

Оптимізація розміру

Стиснення зображень без помітної втрати якості.

Різні розміри з srcset і sizes

Атрибут sizes вказує браузеру, який розмір зображення відображати в залежності від ширини вікна:

  • На мобільних (до 640px) - 100% ширини вікна
  • На планшетах (640-1024px) - 80% ширини вікна
  • На десктопах (понад 1024px) - 50% ширини вікна

Використання CSS функції image-set() для адаптивних фонових зображень із WebP/JPEG альтернативами.
